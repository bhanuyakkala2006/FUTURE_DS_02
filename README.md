# Customer Churn Dashboard

## Project Overview
This project analyzes customer churn data using Excel dashboards and visualizations.

## KPIs
- Churn Rate
- Monthly Charges
- Customer Tenure
- Contract Type

## Key Insights
- Month-to-month customers churn more
- Electronic check users churn more
- High monthly charges increase churn
- Customers with low tenure churn more

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Slicers
